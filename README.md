**Project Overview :**

This repository contains a formal prototype of a website developed as part of a ICTF 0554 Web Technologies team project. The goal was to demonstrate UI/UX design, responsive layouts, or API integration.

**Features :**

- Responsive Design

  Designed product pages using CSS Grid to create clean and structured layouts for browsing skincare products.

- Integrated Skin Analysis Tool

  A step-by-step questionnaire designed to help users identify their specific skin concerns (e.g., dryness, acne) and their overall skin type. Based on user responses, the survey provides an immediate Skin Profile allowing the user to search for the right skincare by searching their skin type in the search bar

- Smart Product Filtering
  
  Skin-Type Search Engine: A dynamic search bar that allows users to filter products by choosing specific skin concerns (e.g., Dry, Oily, Sensitive, Combination).
